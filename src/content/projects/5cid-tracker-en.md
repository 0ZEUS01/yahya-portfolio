### Overview
Developed during a internship at CID, this platform was engineered to replace an inefficient, legacy workflow based on manual Excel sheets. Before this solution, managing complex projects via spreadsheets led to data inconsistencies, communication delays, and outdated information. This web application provides a "single source of truth," ensuring every stakeholder operates on the same real-time data.

### Digital Transformation & Real-Time Awareness
The core value of this system lies in its proactive communication. Unlike static Excel files, this platform features an automated notification engine. Managers receive instant alerts for critical project events such as a task being completed, a phase going out of budget, or any significant change in the project lifecycle. This transition to real-time updates eliminated the risk of miscommunication and allowed for immediate executive intervention when needed.

### Lifecycle & Milestone Management
I engineered a dynamic workflow system that allows teams to update progress instantly. The system is built to handle the complex dependencies inherent in large-scale technical projects, ensuring that milestones are linked to specific budget alerts and delivery timelines. By moving the data to a centralized PostgreSQL database, the platform ensures that no information is lost or overwritten, a frequent issue with the previous spreadsheet-based method.

### Key Features
* **Excel-to-Web Migration:** Successfully digitized fragmented manual tracking into a unified corporate platform.
* **Real-Time Notification Engine:** Instant alerts for managers regarding budget overruns, project completions, and milestone updates.
* **Role-Based Access Control:** Secure permission levels that prevent unauthorized data changes, solving a major flaw of shared spreadsheets.
* **Proactive Budget Monitoring:** Automated tracking that flags financial events as they occur within the project lifecycle.
