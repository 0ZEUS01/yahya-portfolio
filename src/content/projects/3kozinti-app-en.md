### Overview
Developed as a foundational academic project, Kozinti is a social culinary hub where users exchange recipes. The project was built under strict academic constraints, requiring a purely native implementation without the use of external design frameworks.

### Native Craftsmanship
Because this was a academic assignment, every part of the interface was engineered using only the native tools available in **Android Studio**. I manually designed the UI using **XML layouts** and standard components, focusing on creating a smooth user experience through pure Java logic and direct integration with **Firebase**.

### Key Features
* **Social Recipe Exchange:** Share your own culinary secrets and browse recipes from other students and users.
* **Pure Native UI:** Engineered from scratch using native Android XML, showcasing a deep grasp of the core SDK.
* **Interactive Ingredients:** View full lists and step-by-step guides for every community-shared meal.
* **Firebase Backend:** Real-time data syncing and authentication handled directly via Firebase integration.
