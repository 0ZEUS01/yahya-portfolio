### Overview
Developed during a internship at CID, this platform was engineered to modernize corporate file management. It provides a centralized, secure environment for teams to organize and share technical documentation. The final product was successfully deployed and is currently utilized by the organization on their local internal server.

### Hierarchical Access Control
The core of the platform is a sophisticated permission system based on user roles. I implemented logic that allows managers to define folder visibility and document access at a granular level. This ensures that sensitive project files are only accessible to authorized personnel, while managers maintain full administrative control over the directory structure and sharing permissions.

### Technical Workflow
The platform streamlines how departments access critical project data. I engineered a robust backend to handle complex folder hierarchies and implemented an integrated document preview feature. This allows staff to visualize various technical file types directly within the web interface, significantly improving the speed of document review and collaboration.

### Key Features
* **Role-Based Permissions:** Granular access control where managers decide which folders and files are visible to specific user roles.
* **On-Premise Deployment:** Successfully hosted and operational on the organization's local server for maximum data privacy.
* **Integrated Document Preview:** Seamless in-browser visualization of technical files, reducing the need for external software.
* **Enterprise Stack:** A high-performance Full-Stack implementation using Angular and Spring Boot for reliable data handling.
