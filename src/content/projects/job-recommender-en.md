### Overview
LinkMe is an intelligent recruitment ecosystem designed to bridge the gap between job seekers and hiring managers. Developed as part of my academic journey at EMSI, the platform moves beyond simple keyword searching by utilizing Machine Learning to understand the compatibility between a candidate's professional profile and active job openings.

### The Intelligence Engine
The core of the platform is a recommendation engine built with **Scikit-learn**. I implemented professional profiling logic that analyzes user skills and experience to suggest the most relevant career opportunities. To ensure a seamless recruitment loop, the system includes an automated email support module that notifies users of new matches and application statuses, providing a proactive experience for both candidates and recruiters.

### Technical Implementation
The application is built on a high-performance **Flask** backend, managing a complex **MySQL** database for user profiles, job listings, and application tracking. By integrating authentication and job management modules, I created a secure and scalable environment capable of handling the entire recruitment lifecycle from profile creation to final placement.

### Key Features
* **ML-Powered Matching:** Intelligent job recommendations based on skill analysis and user profiles via Scikit-learn.
* **Automated Notifications:** Integrated email support system for real-time updates on applications and new job matches.
* **Complete Career Lifecycle:** Full-featured modules for job posting, profile management, and secure user authentication.
* **Relational Talent Data:** A robust MySQL schema designed to link multi-dimensional candidate data with diverse job requirements.
